# Dunkelmodus Wähler

Ein einfaches, aber leistungsfähiges Tool zum Wechseln zwischen verschiedenen Dunkelmodus-Themen für deine Webseiten und Anwendungen.

## Anforderungen

- Ein moderner Webbrowser
- Grundkenntnisse in HTML/CSS/JavaScript

## Funktionen

- Auswahl aus mehreren vorgefertigten Dunkelmodus-Themen
- Möglichkeit zur Erstellung und Speicherung eigener Farbpaletten
- Einfach zu implementierende API für Entwickler
- Responsive Design

## Installation

1. Klone das Repository:
   ```bash
   git clone https://github.com/deinbenutzername/dunkelmodus-chooser.git
   ```
2. Öffne die `index.html` in deinem Browser.

## Nutzung

Füge das Skript zu deiner Webseite hinzu und rufe die Funktionen auf, um den Dunkelmodus zu aktivieren oder zu deaktivieren!
